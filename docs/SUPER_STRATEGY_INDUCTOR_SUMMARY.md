# 🧠 Super Strategy Inductor - 超强策略归纳器

## 系统概述

**超强策略归纳器**是一个革命性的AI医疗决策系统，能够从FSM路径经验中学习"什么机制路径更好"的规则，并将这些规则融合到下一次agent的策略初始化中。

### 运行端口
- **主系统**: `http://localhost:8512`
- **访问方式**: `streamlit run super_strategy_inductor.py --server.port 8512`

## 🏗️ 核心架构

### 1. FSM + ReplayBuffer 架构
```python
class FSMState:
    """FSM状态表示"""
    - 症状向量 (10维): 腹痛、腹泻、便秘、腹胀、焦虑、抑郁、疲劳、失眠、恶心、食欲不振
    - 上下文向量 (5维): 年龄、性别、病程、严重程度、既往史
    - 总维度: 15维状态空间

class FSMPath:
    """FSM路径表示"""
    - 机制类型: psychological/anti_inflammatory/microbiome/motility/combined
    - 效果评分: 效果×0.4 + (1-副作用)×0.3 + 满意度×0.3

class FSMReplayBuffer:
    """经验缓冲器"""
    - 最大容量: 10,000条经验
    - 经验格式: {state, path, reward, timestamp}
    - 采样功能: 支持批次采样和统计分析
```

## 🎯 四大核心功能模块

### 1. 📊 数据收集模块
**功能**: FSM路径经验的结构化收集
- **患者状态输入**: 症状评分(0-10) + 患者背景信息
- **路径选择**: 多选机制路径（心理调节、抗炎、肠道菌群等）
- **结果记录**: 治疗效果、副作用程度、患者满意度
- **经验存储**: 自动计算综合奖励评分并存入ReplayBuffer

**核心指标**:
- 📈 总经验数: 实时统计
- 🏆 最佳机制: 基于平均奖励自动识别
- 💯 平均奖励: 所有经验的效果评分
- ✅ 成功率: 奖励>0.7的经验占比

### 2. 🧠 规则学习模块
**功能**: 基于经验数据自动归纳策略规则
- **机制效果分析**: 各路径类型的平均效果和成功率对比
- **规则自动归纳**: 
  - 🏆 最优机制路径识别
  - ✅ 高成功率机制识别(>80%)
  - 🎯 稳定性机制识别(效果波动<0.2)
  - 🧠 症状特异性规则(高焦虑→心理路径)
  - 🔥 炎症导向规则(高炎症→抗炎路径)

**学习算法**:
```python
def generate_strategy_rules(stats, buffer):
    """基于统计分析的规则归纳"""
    - 最佳机制: max(mean_reward)
    - 高成功率: success_rate > 0.8
    - 稳定机制: std_reward < 0.2
    - 症状模式: 按症状分组的路径偏好分析
```

### 3. 🎯 策略应用模块
**功能**: 为新患者生成个性化治疗策略

- **智能推荐**: 基于症状模式和历史经验的路径选择
- **相似病例匹配**: 在历史数据中找到相似患者案例
- **预期效果评估**: 基于相似病例的效果预测
