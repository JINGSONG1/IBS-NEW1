## 🚀 使用指南

### 快速启动
```bash
# 1. 克隆或下载项目
git clone https://github.com/yourusername/ReticulotypeToolkit-AI-Medical-Platform.git
cd ReticulotypeToolkit-AI-Medical-Platform

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动所有系统
chmod +x start_all_systems.sh
./start_all_systems.sh

# 4. 访问主系统
# 浏览器打开: http://localhost:8510
```

### 系统管理
```bash
# 检查系统状态
./check_systems.sh

# 停止所有系统
./stop_all_systems.sh

# 查看日志
ls logs/
```
