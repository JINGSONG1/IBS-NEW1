2. **数据字段要求**
```python
必需字段:
- patient_id: 患者ID
- age: 年龄
- gender: 性别 (Male/Female)
- ethnicity: 种族
- baseline_severity: 基线症状严重程度
- improvement_percent: 症状改善百分比

可选字段:
- endometriosis: 子宫内膜异位症 (True/False)
- anxiety_score: 焦虑评分
- rome_iv_subtype: Rome IV亚型
```

3. **使用真实数据验证**
